Correct Chunking Example

Very Clearly Chunked

My goal is to create a website that is legible. That way people can easily understand the point behind the content.

Description

I would like to create a website that gives the user an understanding of why chunking is important. This will definitely make life easier on all of us.

Poor Chunking Example

Everything is so close together.
Unfortunately some people don't understand chunking. When there is too much information, it is already difficult to find the important parts.

Description

When you have a lot of unnecessary information, people can get lost in the mix. I know when I have long articles to read for class or work, I generally try and skim for the important information. If the content is not chunked correctly, web pages can look messy, plain, and overwhelming. It is better to break up the content into more managable chunks in order to portray the important information. If you need people to know something, you need to break it from big paragraphs! Call 9-1-1 for emergencies. Most pepole would miss that prior sentence! Aren't you happy that you read through this entire paragraph? It would be beneficial to everybody to properly chunk their content.
